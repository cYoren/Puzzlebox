# Security Policy

## Supported Versions

The following versions of Puzzle are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us privately.

**Please do not report security vulnerabilities via public GitHub issues.**

### How to report

Please send an email to **khatkarsidhant@gmail.com** with the following information:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact of the vulnerability.
- Any suggested fixes or mitigations.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary. We kindly ask you to practice responsible disclosure and give us time to address the issue before making any information public.

### Appreciation

We appreciate the efforts of security researchers who help keep this project safe for everyone.
